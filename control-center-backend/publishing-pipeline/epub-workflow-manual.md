## Manual ePub Creation Process

### Phase 1: Content Preparation (30-45 minutes)

#### Step 1: Organize Source Materials
```bash
# Create working directory structure
mkdir -p epub-working/
mkdir -p epub-working/content/
mkdir -p epub-working/assets/
mkdir -p epub-working/output/
```

#### Step 2: Convert Markdown to HTML
**Process**: Use Pandoc for HTML conversion
```bash
# Generate clean HTML from markdown
pandoc content.md -o epub-working/content/chapter.html \
    --embed-resources \
    --standalone \
    --metadata title="Chapter Title"
```

#### Step 3: Prepare Cover Image
- **Format**: PNG or JPEG
- **Dimensions**: 1600x2400 pixels (recommended)
- **File size**: Under 2MB
- **Naming**: `cover.png` or `cover.jpg`

#### Step 4: Create Chapter Structure
**Manual Organization**:
- Title page HTML
- Table of contents (auto-generated by Calibre)
- Chapter HTML files in reading order
- Appendices or additional materials

### Phase 2: Calibre Import and Configuration (45-60 minutes)

#### Step 1: Launch Calibre and Create New Book
1. Open Calibre E-book manager
2. Click "Add books" → "Add from HTML files"
3. Select prepared HTML files in reading order
4. Calibre creates initial ePub structure

#### Step 2: Edit Book Metadata
**Required Metadata**:
- **Title**: Publication title
- **Author**: Author name(s)
- **Description**: Book summary
- **Tags**: Category tags for organization
- **Publication Date**: Publication date
- **Language**: Content language (en, ru, etc.)

**Professional Metadata**:
- **ISBN**: If available
- **Series**: If part of a series
- **Publisher**: Publisher information
- **Comments**: Detailed description

#### Step 3: Cover Image Assignment
1. Right-click book in Calibre library
2. Select "Edit book"
3. Navigate to "Images" section
4. Add cover image file
5. Set as cover in book metadata

#### Step 4: Table of Contents Generation
**Manual Process**:
1. In Calibre book editor, select "Tools" → "Table of Contents"
2. Choose "Generate TOC from files"
3. Review and edit chapter titles
4. Adjust hierarchy if needed
5. Apply changes

### Phase 3: Content Review and Formatting (30-45 minutes)

#### Step 1: Text Formatting Review
**Manual Checks**:
- Heading hierarchy consistent
- Paragraph spacing appropriate
- Lists formatted correctly
- Code blocks readable
- Links functional

#### Step 2: CSS Styling (Optional)
**Basic Styling**:
- Font family selection
- Paragraph spacing
- Heading styles
- Code block formatting
- Color scheme (if applicable)

#### Step 3: Internal Structure Validation
1. Navigate through ePub in Calibre viewer
2. Check chapter navigation
3. Verify table of contents links
4. Test internal cross-references

### Phase 4: ePub Generation and Validation (30 minutes)

#### Step 1: ePub Export
1. Select book in Calibre library
2. Click "Convert books"
3. Set output format to "ePub"
4. Configure export settings:
   - Page margins
   - Font embedding
   - Compression level
   - DRM settings (none)

#### Step 2: Quality Validation
**Manual Testing**:
- Open ePub in multiple readers:
  - Calibre viewer
  - Adobe Digital Editions
  - Web browser ePub viewer
- Check formatting consistency
- Verify navigation functionality
- Test on mobile device if available

#### Step 3: Metadata Final Check
- Verify all metadata fields complete
- Check cover image display
- Confirm file size reasonable
- Validate publication information
